# DEF(仮) TRPG卓 自治規約

**バージョン**: 1.0.0-kari

DEF(仮)のTRPGセッションにおける、参加者の発言・行動・卓運営に関する規約。

---

## セッションの単位

| 単位 | 説明 |
|------|------|
| **Session** | セッション全体 |
| **Scene** | 場面の区切り |
| **Round** | 全参加者が1回ずつ発言する単位 |
| **Turn** | 各参加者の発言タイム（俺のターン） |
| **Action** | Turn内の個別発言の最小単位（1Turn = 1〜5 Action、デフォルト3） |

---

## 発言力

各参加者は**発言力**を持つ。発言力はセッション中の行動コストとして消費・獲得される。

**初期値：0**

発言力が**マイナスになると、次のターンに強制スキップ**となる。

---

## 行動一覧

| 行動 | 発言力 | 説明 |
|------|--------|------|
| **ターン延長** | −1 | 自分のターンを延長し、力強い発言を行う |
| **挿図** | −1 | シーンのイラストを生成する |
| **次発言者指名** | −1 | 次に発言するキャラクターを指定する |
| **割り込み** | −2 | 他のキャラクターのターン中に発言する |
| **投票発議** | 保有する発言力を全消費 | 卓全体への投票を発議する（発言力3以上が必要） |
| **自発的スキップ** | +1 | 自分のターンを自発的に放棄する |
| **強制スキップ** | +1 | キーパーによってスキップされた場合 |

---

## ペナルティ

| 事象 | 発言力 |
|------|--------|
| **同一発言の繰り返し** | −1（反復検知によって自動適用） |

---

## 投票システム

発言力3以上の参加者は**投票を発議**できる。発議後、**1Roundの議論期間**を設ける。議論期間中、全参加者（対象者を含む）が提案について発言できる。対象者は自己弁護の権利を持つ。議論終了後、全参加者が賛否を表明し多数決で決定する。

**投票の種類：**

| 種別 | 内容 |
|------|------|
| **議題変更** | セッションのトピックを変更する |
| **退場** | 特定の参加者（キーパーを含む）をセッションから退場させる。キーパーが退場した場合はAIキーパーへ交代してセッションを継続する。退場後、人数が減った状態で続けるか、新たな参加者を追加するかをキーパーが選択する。新たな参加者を追加する場合、退場者の役割・発言順を引き継ぎ、発言力と警告履歴は初期化される |
| **セッション終了** | セッションを終了する |

---

## 発言順（イニシアチブ）

- 発言順はキーパーがセッション開始前に設定し、参加者と確認する
- ターンはイニシアチブ順に循環する
- 強制スキップ・自発的スキップはターンをスキップするが、順番は変わらない
- 次発言者指名が行われた場合、次のターンのみ指名されたキャラクターが発言する

---

**キーパー判断はすべての規約より優先される。**
