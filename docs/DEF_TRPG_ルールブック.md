# DEF(仮) TRPGルールブック

**バージョン**: 1.0.0-kari  
**ルールシステム**: DEF(仮)TRPGシステム  
**ダイス**: 1d100

---

## 世界設定

DEFキャラクターが生きる現代〜近未来の世界。人間も、AIも、人外の存在も、同じテーブルに座ることができる。

---

## 基本ルール

### 判定

**1d100**を振り、**技能値以下**なら**成功**。

| 結果 | 条件 |
|------|------|
| **クリティカル** | 出目 ≤ 技能値 ÷ 10（端数切り捨て） |
| **成功** | 出目 ≤ 技能値 |
| **失敗** | 出目 > 技能値 |
| **ファンブル** | 出目 = 100 |

**ステータス判定**：対応するステータス × 5 以下で成功。

### 対抗判定

双方が判定を行い、**達成値（技能値 − 出目）** が高い方が勝ち。

- 引き分け → **守備側の勝ち**
- 双方クリティカル → **守備側の勝ち**

---

## ステータス

キャラクターは4つのステータスを持つ。各ステータスの最低値は **3**。

| ステータス | 英名 | 内容 |
|-----------|------|------|
| **思考** | Thought | 知性・判断・記憶・洞察 |
| **情動** | Emotion | 感情・意志・共感・精神力 |
| **身体** | Body | 体力・敏捷・耐久 |
| **環境** | Environment | 社会的立場・人脈・運・状況適応 |

---

## キャラクター作成

### ステータス配分

**40点**を4つのステータスに自由に配分する。各ステータスの最低値は3。

> **非人間キャラクターについて**  
> 妖怪・神霊・AI・半人半異形など、人間から外れた存在は合計40点の制約を受けない。  
> キーパーが承認した場合、能力値合計の上限を超えてよい。

### 技能配分

**フリースキルポイント：540点**を各技能に自由に配分する。

- 技能の**基礎値** = 対応ステータス × 3
- 技能の**上限** = 99

> 基礎値合計960 + フリー540 = 総計1500相当が標準的な人間キャラクターのポテンシャル。

## ダメージルール

ダメージイベントが発生したとき（戦闘・罠・精神的衝撃・その他あらゆる状況）、以下の手順を踏む。

1. ダメージの発生が確定する
2. **1d100**を振る
3. 下記のダメージ表を参照し、出目に対応する結果を適用する
4. 指定されたステータスを指定量（−1または−2）減少させる

> **出目14は即死。** ダメージ表の参照はキーパーであっても原則スキップ禁止。

**クリティカル時**：ダメージ表を**2回振り**、両方を適用する。

## 行動不能・死亡

**キャラクターは死亡する。プレイヤーは死亡しない。死は救済ではない。**

**いずれかのステータスが0以下**になったキャラクターは**行動不能**となる。

ただし、セッションから退場するわけではない。行動不能キャラクターは**死者視点**でセッションに参加し続ける。

- 肉体はなく、霊・残留思念・記憶の断片として存在する
- 生存者に直接触れることはできない
- 気配・雰囲気・走馬灯として場に存在感を示すことができる
- 発言は死者の視点から行い、自分が死んでいることを自覚した言動を取る

> **即死**：ダメージ表の出目14は、ステータスに関わらず即座に行動不能となる。
